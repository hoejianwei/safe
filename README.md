# safe
safe for escape room, trigger qlab cue when players unlock, disarm safe to prevent retriggering, allow server to send cue to arm safe again
feel free to adapt it for whatever u need to

Parts used

XIAO ESP32-C6
Time of flight VL53L0X

TOF is far more reliable than ultrasonic
Requires reboot after upload, if not it will trigger automatically for some reason

pics and video available as part of documentation
